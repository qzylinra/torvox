# P2.4 + P2.5 实现规范

> 字体 + 主题 + 设置屏幕

## 概述

P2.4 实现字体大小调整、字体选择、主题系统和 TrueColor 支持。
P2.5 实现 Jetpack Compose 设置屏幕、Shell 选择、DataStore 持久化。
两者紧密耦合（设置控制字体/主题），因此合并实现。

---

## P2.4: 字体 + 主题

### F1: 字体大小调整

**实现路径**: 
- Kotlin: pinch-to-zoom 手势 on TerminalSurface → ViewModel → Rust bridge
- Rust: `FontPipeline::set_font_size(f32)` 清除 glyph cache → 重新计算 cell_size
- GPU: `GpuUniforms.cell_size` 更新

**API**:
```rust
// bridge.rs
#[boltffi::export]
fn update_font_size(session_id: i64, size: f32) -> Result<(), TerminalError>
```

### F2: 内置主题

**主题结构** (Rust):
```rust
pub struct Theme {
    pub name: String,
    pub bg: [u8; 3],
    pub fg: [u8; 3],
    pub cursor: [u8; 3],
    pub ansi: [[u8; 3]; 16],  // 16 ANSI colors
}
```

**内置主题** (Kotlin data objects):
- Catppuccin Mocha (default)
- Dracula
- Solarized Dark / Light
- Nord
- Tokyo Night
- Gruvbox Dark
- One Dark
- Monokai
- GitHub Dark
- Rosé Pine

### F3: TrueColor 支持

已存在: VT parser `\e[38;2;R;G;Bm` → `Color::new(r,g,b)` → Cell.fg → GPU fg_color。
验证: `sgr_truecolor_fg` 和 `sgr_truecolor_bg` 测试已通过。

## P2.5: 设置屏幕

### S1: 设置屏幕布局

```
SettingsScreen
├── Section: 外观
│   ├── Font Size (Slider: 8-32, default 14)
│   ├── Font Family (Dropdown: system monospace fonts)
│   └── Theme (Grid: built-in themes with preview)
├── Section: 终端
│   ├── Shell (Dropdown: /system/bin/sh, bash, zsh, fish)
│   ├── Scrollback Lines (Slider: 1000-100000)
│   └── Cursor Style (Block/Underline/Bar)
├── Section: 输入
│   ├── Touch Behavior (Radio: Right click/Middle click/None)
│   └── Modifier Keys (Toggle: which keys to show)
└── Section: 关于
    ├── Version
    └── GitHub link
```

### S2: DataStore 持久化

**依赖**: `androidx.datastore:datastore-preferences:1.1.7`

**SettingsRepository**:
```kotlin
class SettingsRepository @Inject constructor(
    private val dataStore: DataStore<Preferences>
) {
    val fontSize: Flow<Int>
    val fontFamily: Flow<String>
    val themeName: Flow<String>
    val shell: Flow<String>
    val scrollbackLines: Flow<Int>
    // ...
}
```

### S3: 导航

- TerminalScreen 顶部 App Bar 添加 Gear 图标
- 点击 Gear → 导航到 SettingsScreen
- SettingsScreen 返回按钮 → 回到 TerminalScreen
- 使用简单 `NavHost` 或 `AnimatedContent`

## 实现顺序

1. **Rust: Wire RenderConfig** → bridge.rs 添加 theme/font_size 方法
2. **Rust: Theme struct** → torvox-core 添加 Theme 类型
3. **Kotlin: Theme system** → 创建 Theme.kt + 内置主题
4. **Kotlin: DataStore** → 添加依赖 + SettingsRepository
5. **Kotlin: SettingsScreen** → Compose 设置界面
6. **Kotlin: Navigation** → 添加路由
7. **Kotlin: Font size gesture** → pinch-to-zoom
8. **Kotlin: Wire settings** → 读取设置应用到 TerminalConfig
9. **测试** → 编译 + 全部测试
